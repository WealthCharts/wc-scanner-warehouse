### create a new environment
```bash
virtualenv venv
```
#### or

```bash
py -3.9 -m venv env
```

### activate the environment
```bash
venv\Scripts\activate
```

### install dependencies
```bash
pip install -r requirements.txt
```

### freeze the environment
```bash
pip freeze > requirements.txt
```

### run the application
```bash
python app.py
```
